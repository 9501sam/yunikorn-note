## deployment
* ```nginx.yaml```
```
apiVersion: apps/v1
kind: Deployment
metadata:
  labels:
    app: nginx
  name: nginx
spec:
  replicas: 1
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
        applicationId: "nginx_2019_01_22_00001"
        queue: root.sandbox
      name: nginx
    spec:
      schedulerName: yunikorn
      containers:
        - name: nginx
          image: "nginx:1.11.1-alpine"
          resources:
            requests:
              cpu: "500m"
              memory: "1024M"
```
```
$ kubectl apply -f nginx.yaml -n yunikorn
$ kubectl get pod -n yunikorn
```
